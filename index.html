Portfolio

HTML

HyperText Markup Language hipertekstowy język znaczników, obecnie jest szeroko wykorzystywany do tworzenia stron interenetowych.

CSS

Cascading Style Sheet to język służący do opisu formy prezentacji (wyświetlania) stron WWW. CSS został opracowany przez organizację W3C w 1996r.

W3C

World Wide Web Consortium to organizacja, która zajmuje się ustanawianiem standardów przesyłu stron WWW. Została założona 1 października 1994 roku przez Tima Berners-Lee, twórcę WWW oraz autora pierwszej przeglądarki internetowej i serwera WWW.

INNE

Dalsze techniki rozwoju strony WWW.
